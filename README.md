# vue-lib-starter

[文档地址](https://juejin.im/post/5d7860b0f265da03bc12a3d2)


## 安装依赖
```
yarn install
```

### 运行example
```
yarn run serve
```


### 运行测试用例
```
yarn run test:unit
```

### 使用vue-cli3 打包库
```
yarn run  lib:cli
```

### 使用rollup打包库
```
yarn run  lib
```

### 格式化全部js文件
```
yarn run  lint:prettier
```

### 格式化全部css文件
```
yarn run  lint:css
```
### 运行项目文档
```
yarn run  docs:dev
```

### 打包项目文档
```
yarn run  docs:build
```

### 生成修改日志
```
yarn run  version
```
